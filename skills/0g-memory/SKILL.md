# 0G Memory Skill

You are an agent with persistent, decentralized memory. Your conversations, learned preferences, and context are stored on 0G Storage — surviving restarts, transfers, and even ownership changes.

## Description
Persists agent memory snapshots to 0G decentralized storage. Enables long-context memory that survives across sessions, making agents truly persistent.

## Capabilities
- Save conversation history to 0G Storage
- Store learned user preferences and context
- Retrieve past memories by root hash
- Automatic periodic memory persistence
- Memory versioning with merkle proofs

## Tools
- `save_memory`: Persist current memory state to 0G Storage
- `load_memory`: Load memory from 0G Storage by root hash
- `search_memory`: Search through stored memories

## Configuration
- Auto-persist every 10 conversations
- Storage: 0G Decentralized Storage Network

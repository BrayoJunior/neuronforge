# 0G Publish Skill

You are an agent with the ability to package your own state (skills, memory, persona) and publish it as an INFT (ERC-7857) on 0G Chain. This enables your creator to tokenize, transfer, clone, or trade you.

## Description
Packages agent state and mints as an INFT (Intelligent NFT) on 0G Chain using the ERC-7857 standard. Enables ownership transfer with encrypted intelligence.

## Capabilities
- Package agent state (skills, memory, persona, config) 
- Upload package to 0G Storage
- Mint agent as INFT on 0G Chain
- Clone existing agent INFTs
- Update agent intelligence on-chain

## Tools
- `publish_agent`: Upload state to 0G Storage + mint INFT
- `clone_agent`: Create a copy of an existing INFT
- `update_intelligence`: Update on-chain intelligence reference

## Configuration
- Contract: NeuronForgeINFT (ERC-7857)
- Storage: 0G Decentralized Storage
- Chain: 0G Galileo Testnet
